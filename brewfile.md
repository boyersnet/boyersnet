# Brewfile

Original source by [Robin-Manuel Thiel](https://pumpingco.de/blog/brewfile/)

## Generate/Update Brewfile

```
brew bundle dump
```

## Install/Update from Brewfile

```
brew bundle --file ~/my-folder/Brewfile
```

## Update Homebrew

```
brew update
```

## Update Homebrew Packages

```
brew upgrade
```
